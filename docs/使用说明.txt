风扇灯控制 - 使用说明 / User Guide
====================================

通过 OpenRGB 控制主板 ARGB 灯光，支持主流品牌主板。
Controls motherboard ARGB lighting via OpenRGB. Supports all major brands.

使用步骤 / Steps:
------------------

1. 安装 Python 3（如未安装），从 https://python.org 下载
   Install Python 3 from https://python.org (check "Add Python to PATH")

2. 双击「start.bat」即可 / Double-click 「start.bat」to launch

   程序会自动完成以下操作 / The program will:
   - 安装 openrgb Python 包 / Install openrgb package
   - 下载 OpenRGB（如未找到）/ Download OpenRGB if missing
   - 启动 OpenRGB 服务 / Start OpenRGB server
   - 连接并控制灯光 / Connect and control lighting

3. 开机自启 / Auto-start on boot
   勾选界面中的「Auto start on boot」即可
   Check the "Auto start on boot" checkbox in the UI
